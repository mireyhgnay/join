# 회원가입 & 로그인 페이지

## Setting

**node v20.18.3**

<br />

**install**

```bash
npm i
```

<br />

**개발 모드 빌드 및 실행**

```bash
npm run dev
```

<br />

**프로덕션 모드 빌드 및 실행**

```bash
npm run build
npm run start
```

<br />

**lint 검사**

```bash
npm run lint
```
